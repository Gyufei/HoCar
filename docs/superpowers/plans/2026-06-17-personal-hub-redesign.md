# Personal Hub Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert the app into a lightweight personal admin platform with a shadcn/ui-based login page, authenticated sidebar layout, dashboard home, and migrated bills module pages.

**Architecture:** Keep Next.js App Router and Auth.js. Add shadcn/ui as the component foundation, then create a reusable authenticated app shell under a route group while leaving login outside the shell. Move bills pages into `/bills/*` and keep legacy route redirects.

**Tech Stack:** Next.js, React 19, Tailwind v4, Auth.js / NextAuth, Prisma, shadcn/ui, TypeScript.

---
